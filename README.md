### O que eu aprendi :shipit::
- **Translate**:
    ```CSS
    .exemplo {
        transform: translate(15px, 20px)
    }
    ```
    Permite que o elemento seja deslocado. No exemplo acima o elemento será deslocado 15px para cima e 20px para a direita.
